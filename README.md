# Example for 2025 Reinterpretation Forum Workshop

`pixi` example using HEP Packaging Coordination conda-forge tooling for the [2025 Reinterpretation Forum Workshop](https://indico.cern.ch/event/1466101/contributions/6287842/)

## Getting Started

1. Install [`pixi`](https://pixi.sh/)
2. Clone this repository and navigate to its top level.
3. Run `pixi shell`
